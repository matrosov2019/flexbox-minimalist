### flexbox-minimalist
`npm i flexbox-minimalist --save`

### How to import
import 'flexbox-minimalist/dist/flexbox.css';

### Example
```html
<div class="flex flex-auto flex-col">
    <div class="flex flex-none">top</div>
    <div class="flex flex-auto"></div>
    <div class="flex flex-none">bottom</div>
</div>
```
